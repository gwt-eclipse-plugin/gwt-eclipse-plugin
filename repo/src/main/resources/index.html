Plugin installation

To install the GWT Eclipse Plugin:

  1. In Eclipse, click on Help -> Install New Software... -> Add...

  2. (The "Add Repository" dialog will appear)

  3. For "Name", enter "GWT Eclipse Plugin" or something similar

  4. For "Location", fill-in the URL of this web page (https://plugins.gwtproject.org/eclipse/gwt-eclipse-plugin/4.1.0/)
